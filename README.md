# Digit_Recognition_ml
This project is a simple handwritten digit recognition project built using Neural Network.
It uses MNIST dataset which contains images of digits from 0 to 9.
The model is trained to classify handwritten digits with high accuracy using TensorFlow and Keras.

The project uses the MNIST dataset, which includes:  
60,000 training images  
10,000 testing images  
Each image is 28x28 grayscale pixels.
